## Steps to Contribute

### 1. Fork the Repository
Click the *Fork* button and clone your fork:
bash
git clone https://github.com/your-username/ChatGenz.git

2. Create a Branch

git checkout -b feature/my-update

3. Make Your Changes

Write clean, readable, console-based C# code.

4. Commit Your Work

git add .
git commit -m "Describe your change here"

5. Push + Pull Request

git push origin feature/my-update

Then open a Pull Request from GitHub.


---

Guidelines

Must remain console-based (GDB-friendly)

No external libraries

Keep features simple and readable

Document new features in README.md
